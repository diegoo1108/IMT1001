# IMT1001: Introducción a Ingeniería Matemática
El Ramo IMT1001 es un ramo opcional para la licenciatura de ingeniría matemáticas en la Pontificia Universidad Católica. Este tiene como objetivo mostrar distintas técnicas y tareas comúnmente asociada a este major. 

Se cursó este ramo el primer semestre del 2023 cuando me encontraba en mi tercer semestre de licencitura.

## Proyectos
Este curso corresponde a 6 tareas individuales y un proyecto final grupal. A continuación se desglosan estas tareas, sus objetivos y comentarios.

### <code style='color: gold'>Tarea 1:</code> Page Rank
Implementación del algoritmo de Page Rank en python.
### <code style='color: gold'>Tarea 2:</code> Big Data
Clasificadores k-mean y Decision Trees.
### <code style='color: gold'>Tarea 3:</code> Teoría de Computación
Programar una máquina de Turing.
### <code style='color: gold'>Tarea 4:</code> Manejo de Información
Webscraping y análisis de sentimiento.
### <code style='color: gold'>Tarea 5:</code> Problemas Inversos
Implementar la transfromada de Radón y sintetizar una imagen tomográfica.
### <code style='color: gold'>Tarea 6:</code> Análisis Numérico
Resolver ecuaciones diferenciales parciales usando métodos numéricos.
